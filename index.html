Project One: Mom's Food Delicacies

Description: Mom's Food Delicacies is a comprehensive website designed to enhance the online presence of a food business. It includes features such as an admin panel for managing stock, processing orders, and tracking inventory. Administrators can easily update product listings, monitor stock levels, and handle supplier information through an intuitive interface.

Technologies Used: HTML, CSS, JavaScript, Bootstrap, PHP, MySQL.

Key Features:

  • Responsive design for mobile and desktop devices.
  • Secure admin panel with authentication.
  • Real-time stock and inventory updates.
  • Customer order tracking system.
Project Two: Mom's Food Delicacies WUI

Description: This desktop application is built using Visual Basic (VB) and Windows Forms Application (WFA) framework. It aims to streamline the operations of a small food business by managing stock levels, tracking orders, and facilitating customer interactions efficiently.

Technologies Used: Visual Basic, Windows Forms Application, SQL Server.

Key Features:

  • User-friendly interface for easy navigation.
  • Real-time stock management and inventory tracking.
  • Order placement and tracking system.
  • Customer management and interaction logs.
